LEGEND:
• Row / Col = physical matrix position on Haas panel
• Output = key or function sent via USB HID
• “Shift+”, “Ctrl+”, or “Alt+” prefixes = modifier behavior
• Keys beyond F24 are virtual assignments (software-defined in LinuxCNC / Probe Basic)
• Mappings now reflect tested shortcut integrations (Rapid, Feed, Spindle overrides, etc.)
