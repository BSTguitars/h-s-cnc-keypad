LEGEND:
• Row / Col = physical matrix position on Haas panel
• Output = key or function sent via USB HID (Ctrl+Alt combos)
• “Ctrl+Alt+” prefixes = protected printable combinations
• No F-keys or virtual keys are used
• Reflects tested bindings for Probe Basic (Rapid, Feed, Spindle, Coolant, etc.)
